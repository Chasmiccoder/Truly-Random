Usage

First select the type of random number you want to generate.
Coin Toss Simulates a 'Quantum Coin Toss', and gives either Tails (0) or Heads (1)
Integer Generates integers between the specified lower and upper limits
Float Generates floating point numbers between the specified lower and upper limits
N Digit Integer Generates integers with N digits
Fraction Generates a float between 0 and 1
N Digit Binary Sometimes you just want a binary string of N digits xD

The parameter 'How Many' specifies the number of random numbers you want to produce in a single query.
The parameter 'Precision' specifies the number of digits after the decimal place during generation of floating point numbers.
